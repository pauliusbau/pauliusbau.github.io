Laiko rėžis datos rėžio oracle užklausoje

Date tr 27 rugpjutis 2014 Tags sql / date range / oracle / time interval / time period / time range / week days / weekend

Laikas nuo laiko tenka ištraukti tam tikrų valandų kelių dienų duomenis, pav. gauti duomenis už praėjusio mėnesio kiekvieną dieną 08:00–09:00 valandomis. Kartais pasitaiko dar įdomesni poreikiai, pav. gauti tam tikro laiko intervalo duomenis tik savaitgaliais arba tik konkrečią dieną (pav. tik antradieniais ir pan.). Užduotis paprasta, bet dėl to, kad tai daryti tenka ne dažnai, vis pamirštu konkrečią sintaksę. Tenka pasikelti senus failus, juose ieškoti konkretaus sprendimo. Taip daryti nusibodo ir nusprendžiau savo naudojamą techniką aprašyti čia. Gal kam nors irgi pravers...

Tarkim, turim lentelę t, kurioje yra saugomi maždaug mėnesio date tipo įrašai, kas 15 minučių. Pasibandymui (pav. ant sqlfiddle.com) tokią lentelę galit pasigaminti taip:

 1  create table t
 2  (id number, DT date);
 3
 4  insert into t (id, DT)
 5  SELECT
 6  rownum as id
 7  ,TO_DATE('2014-08-01', 'yyyy-mm-dd') + (rownum-1)*(15/24/60) AS DT
 8  FROM dual
 9  CONNECT BY LEVEL <= (TO_DATE('2014-08-27 00:00:00', 'yyyy-mm-dd HH24:MI:SS') - TO_DATE('2014-08-01 00:00:00', 'yyyy-mm-dd HH24:MI:SS'))*24*4
10  order by 1;

Pačiam paprasčiausiam duomenų ištraukimui galima panaudoti tokį selectą:

1  select
2  ID
3  ,TO_CHAR(DT, 'yyyy-mm-dd HH24:MI:SS') as DT
4  from t
5  order by ID

Select sakinio rezultatas yra maždaug toks:

10  2495 2014-08-26 23:30:00
11  2496 2014-08-26 23:45:00

Pabandyk šiuose 2496 įrašuose atsirinkti 2014-08-01 – 2014-08-03 dienų "duomenis" nuo 08:00 iki 09:00. Tam prieš tai buvusi selectą papildom taip:

1  select
2  ID
3  ,TO_CHAR(DT, 'yyyy-mm-dd HH24:MI:SS') as DT
4  from t
5  where
6  TO_CHAR(DT, 'yyyy-mm-dd') between '2014-08-01' and '2014-08-03'
7  and TO_CHAR(DT, 'HH24:MI:SS') >= '08:00:00'
8  and TO_CHAR(DT, 'HH24:MI:SS') <= '09:00:00'
9  order by ID

Dabar pabandykim išsitraukti 2014-08-01 – 2014-08-15 periodo savaitgalių duomenis nuo 08:00 iki 09:00. Tam prieš tai buvusi selectą papildom taip:

 1  select
 2  ID
 3  ,TO_CHAR(DT, 'yyyy-mm-dd HH24:MI:SS') as DT
 4  from t
 5  where
 6  TO_CHAR(DT, 'yyyy-mm-dd') between '2014-08-01' and '2014-08-15'
 7  and TO_CHAR(DT, 'dy') in ( 'sat','sun' )
 8  and TO_CHAR(DT, 'HH24:MI:SS') >= '08:00:00'
 9  and TO_CHAR(DT, 'HH24:MI:SS') <= '09:00:00'
10  order by ID

Panašiu principu galim užklausi duomenų visoms darbo dienoms, t. y. be savaitgalių:

 1  select
 2  ID
 3  ,TO_CHAR(DT, 'yyyy-mm-dd HH24:MI:SS') as DT
 4  from t
 5  where
 6  TO_CHAR(DT, 'yyyy-mm-dd') between '2014-08-01' and '2014-08-15'
 7  and TO_CHAR(DT, 'dy') not in ( 'sat','sun' )
 8  and TO_CHAR(DT, 'HH24:MI:SS') >= '08:00:00'
 9  and TO_CHAR(DT, 'HH24:MI:SS') <= '09:00:00'
10  order by ID
